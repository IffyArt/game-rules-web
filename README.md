# 武士道 TCG 應用大會規則 - 中文翻譯與專業歸類版
### (Bushiroad TCG Advanced Floor Rules ver. 1.2.11 - Traditional Chinese Translation)

本專案將武士道官方最新發布的 **《ブシロードTCG応用フロアルール ver.1.2.11》**（適用開始日：2025年12月10日）進行了專業的繁體中文翻譯與結構化歸類。

翻譯標準基於**競技卡牌遊戲玩家與裁判的專業術語**（如瑞士輪、差負、失格、漏效果、多抽牌等），且在每個段落與章節末尾標註了 **`[原書第 X 頁]`**，方便讀者與日文原版 PDF 進行快速對照與勘誤。

## 🎯 專案特色

1.  **玩家語境優先：** 本譯文捨棄了生硬的直譯，全面對齊目前主流 TCG 社群的玩家口頭禪與競技裁判學名詞（例如將「自動能力のプレイ忘れ」譯為「漏掉自動能力/漏效果」，將「手札にカードを余分に加えてしまった」譯為「多抽牌/額外摸牌」）。
2.  **精準對比：** 每節末尾附帶的 `[原書第 X 頁]` 標記均經過仔細核對，便於玩家在遇到爭議時，第一時間回溯日文 PDF 尋求裁判依據。
3.  **無用資訊過濾：** 本文件聚焦於目前仍在營運的所有 11 款現行遊戲，自動過濾了原 PDF 結尾中針對已停運舊遊戲（如 Monster Collection、Luck & Logic、Victory Spark 等）的過期參考資料，使規則書更加清晰實用。
