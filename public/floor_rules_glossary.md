# 專業卡牌遊戲裁判與玩家術語對照表 (TCG Glossary)

本術語表旨在為**武士道 (Bushiroad) TCG 應用裁判與大會規則 (Advanced Floor Rules)** 的翻譯提供統一的專業名詞標準。所有譯名均基於華語圈主流競技卡牌遊戲（如 Magic: The Gathering、Yu-Gi-Oh!、Weiss Schwarz、Shadowverse EVOLVE 等）的通用學術與玩家慣用語。

---

## 1. 大會角色與職責 (Roles & Responsibilities)

| 日文原文 | 專業中文譯名 | 英文/通用語 | 備註 |
| :--- | :--- | :--- | :--- |
| ファイター | **玩家 / 選手 / 鬥士** | Fighter / Player | 本文中統一譯為「玩家」以符玩家習慣，涉及具體遊戲氛圍時亦可稱「選手/鬥士」。 |
| 主催者 | **主辦方 / 主辦者** | Organizer (TO) | 負責大會運營的實體或個人。 |
| 運営スタッフ | **工作人員 / 運營人員** | Tournament Staff | 負責登記、廣播、錄入成績等行政工作的非裁判人員。 |
| ジャッジ | **裁判** | Judge | 負責執法、處理違規並維持遊戲秩序。 |
| ヘッドジャッジ | **主裁判 / 總裁判** | Head Judge (HJ) | 賽事的最終裁決者，擁有最終解釋權與裁決權。 |
| 観戦者 | **旁觀者 / 觀眾** | Spectator | 在場外觀看比賽的群眾。 |
| プレス | **媒體 / 記者** | Press / Media | 負責採訪與攝影的人員。 |

## 2. 牌組與遊戲配件 (Decks & Accessories)

| 日文原文 | 專業中文譯名 | 英文/通用語 | 備註 |
| :--- | :--- | :--- | :--- |
| デッキ | **牌組 / 卡組** | Deck | 玩家參賽的主牌組。 |
| デッキレシピ | **牌表 / 牌組登記表** | Decklist / Deck Recipe | 玩家在賽前提交的牌組配置清單。 |
| トークン | **衍生物 / Token** | Token | 效果產生的臨時實體卡牌。 |
| マーカー | **標記 / 指示物** | Marker | 用於標記遊戲狀態（如能量、天賦等）的物件。 |
| スリーブ | **卡套** | Sleeve | 保護卡牌的塑料套。 |
| プレイマット | **卡墊 / 對戰墊** | Playmat | 鋪在桌面上的遊戲專用軟墊。 |
| 補助アイテム | **輔助道具 / 計數器** | Counters / Auxiliary Items | 骰子、指示物、算血器等。 |
| ダイス（さいころ） | **骰子** | Dice | 一般指六面骰。 |
| 特殊なカード | **特殊卡 / 附加卡** | Extra/Special Cards | 雙面卡、特製指示物卡等非主牌組卡牌。 |

## 3. 洗牌與卡牌操作 (Shuffling & Manipulation)

| 日文原文 | 專業中文譯名 | 英文/通用語 | 備註 |
| :--- | :--- | :--- | :--- |
| シャッフル | **洗牌** | Shuffle | 將牌組隨機化的動作。 |
| カット | **切牌** | Cut | 將牌組分成數疊並重新組合的簡易隨機動作。 |
| ディールシャッフル | **堆疊洗牌 / 攤牌** | Pile Shuffle / Deal Shuffle | 將牌組分發成複數堆疊的洗牌法，常用於數張數。 |
| オーバーハンドシャッフル | **切牌式洗牌 / 印度洗牌** | Overhand Shuffle | 常見的手持牌組頂部向下滑動洗牌法。 |
| リフルシャッフル | **完美洗牌 / 彈牌洗牌** | Riffle Shuffle | 雙手將卡牌對插的洗牌法（可能對卡牌造成物理損耗）。 |
| 代理カット | **代理切牌** | Cut-by-proxy | 由對手委託裁判或第三方代為切牌的動作。 |
| ハンドシャッフル | **切手牌 / 切牌** | Hand Shuffle | 玩家手中卡牌頻繁交換位置的習慣動作。 |

## 4. 賽事制度與配對 (Tournament Formats & Pairings)

| 日文原文 | 專業中文譯名 | 英文/通用語 | 備註 |
| :--- | :--- | :--- | :--- |
| スイスドロー | **瑞士輪 / 瑞士制** | Swiss Draw | 積分類似者相互配對的賽制。 |
| ダブルエリミネーション | **雙敗淘汰制** | Double Elimination | 輸掉兩場即被淘汰的賽制。 |
| シングルエリミネーション | **單敗淘汰制 / 淘汰賽** | Single Elimination | 輸掉一場即淘汰的賽制。 |
| マッチ | **對局 / Match** | Match | 完整的比賽單位（如三局兩勝 Bo3 或一局一勝 Bo1）。 |
| セット / ゲーム | **局 / 盤 / Game** | Game / Set | Match 中的單次對戰。 |
| ブシナビ | **Bushinavi** | Bushinavi | 武士道官方的賽事 App。 |
| 順位の算出 | **同分排名計算 / 小分** | Tie-breakers / Standings | 決定積分相同時的名次方法。 |
| 対戦相手の平均勝率 | **對手平均勝率 (OMW%)** | Opponent Match Win Percentage | 衡量對手強度的指標。 |
| 自身のゲーム勝率 | **個人勝率 (GW%)** | Game Win Percentage | 玩家在所有單局中的勝率。 |
| 対戦相手の平均ゲーム勝率 | **對手平均單局勝率 (OGW%)** | Opponent Game Win Percentage | 玩家所有對手的平均單局勝率。 |

## 5. 比賽行為與結果 (Match Results & Actions)

| 日文原文 | 專業中文譯名 | 英文/通用語 | 備註 |
| :--- | :--- | :--- | :--- |
| 投了 | **認輸 / 投降** | Concede | 玩家主動宣告此局/此對局落敗。 |
| 合意による引き分け | **意向平局 / 協議平局 (ID)** | Intentional Draw (ID) | 雙方協議以平局收場。 |
| リタイア / 途中棄権 | **退賽 / 棄權 / 退席** | Drop / Withdraw | 玩家不再參與後續所有輪次的比賽。 |
| エクストラターン | **額外回合 / 加時回合** | Extra Turns | 比賽時間截止後的追加回合。 |
| チャレンジ（再確認） | **申訴 / 二次確認 / 挑戰** | Appeal | 對一般裁判裁定不服，向主裁判申請重新判定。 |

## 6. 違規與判罰 (Infractions & Penalties)

| 日文原文 | 專業中文譯名 | 英文/通用語 | 備註 |
| :--- | :--- | :--- | :--- |
| 違反行為 | **違規 / 違反規則** | Infraction | 玩家非故意違反規則的行為。 |
| 不正行為 | **作弊 / 不正當行為** | Cheating | 故意的、試圖獲取不正當優勢的違規行為。 |
| 罰則 | **罰則 / 判罰** | Penalty | 裁判針對違規做出的正式判罰。 |
| 注意 | **注意** | Caution | 最輕微的口頭糾正與記錄。 |
| 警告 | **警告** | Warning | 正式警告，累積可能升級為更重判罰。 |
| ゲームの敗北 | **局負 / 盤負** | Game Loss | 直接判處當前局/盤落敗。 |
| マッチの敗北 | **差負 / 對局負** | Match Loss | 直接判處當前對局（Match）落敗。 |
| 失格 | **取消資格 / DQ** | Disqualification (DQ) | 驅逐出賽，情節嚴重者面臨停賽調查。 |

## 7. 常見違規類型 (Common Infractions)

| 日文原文 | 專業中文譯名 | 英文/通用語 | 備註 |
| :--- | :--- | :--- | :--- |
| 不適正なデッキ / レシピ | **非法牌組 / 牌表不符** | Illegal Deck / Decklist Error | 牌組或牌表配置不合規則。 |
| 区別のつくカード・スリーブ | **標記卡 / 標記卡套** | Marked Cards/Sleeves | 卡牌或卡套有物理磨損或標記。 |
| 誤った情報を伝えた | **傳遞錯誤資訊** | Miscommunication | 向對手或裁判提供不實遊戲狀態。 |
| ゲームの状態が不適正になった | **遊戲狀態非法** | Illegal Game State | 由於誤操作導致遊戲狀態無法正常回溯。 |
| 自動能力のプレイ忘れ | **漏效果 / 漏掉自動能力** | Missed Trigger | 忘記觸發必發的自動能力。 |
| 見てはいけないカードを見た | **偷看卡牌 / 看外卡** | Looking at Extra Cards | 窺視了圖書館或對手手牌中不公開的卡。 |
| 手札にカードを余分に加えた | **多抽牌 / 額外摸牌** | Drawing Extra Cards (DEC) | 摸取了超過效果規定數量的牌至手牌。 |
| スロープレイ | **慢玩 / 緩慢遊戲** | Slow Play | 非故意的遊戲节奏過慢，消耗比賽時間。 |
| 非紳士的行為 | **非紳士行為 / 言行不當** | Unsporting Conduct | 辱罵對手、砸牌、摔配件等行為。 |
| 遅刻 | **遲到** | Tardiness | 輪次開始時未準時到着座位。 |
| 連続行動の手順の変更 / 短縮 | **連續行動簡化 / 捷徑** | Shortcuts | 雙方認可的遊戲操作簡化（如檢索完直接宣告下一步）。 |
